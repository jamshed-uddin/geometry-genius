What is event bubbler?

Event bubbling is a term in DOM,when an element recieves an event it bubble up the element to its parent and upper element.It relates to the order in which event handlers are called when one element is nested inside a second element, and both elements have registered a listener for the same event.For example there is container div,another inner container div inside container div,then a button inside.If we add a event handler to button and we are also clicking it's parent element.That's how it bubble up the event to the parent elements.

What is callback function? What is the purpose of callback function?

In JavaScript we can pass object as parameter of an function.We also can pass a function as another function's parameter.
Callback function make sure that a function is not going to run before a task is completed but will run right after the task has completed. It helps us develop asynchronous JavaScript code and keeps us safe from problems and errors.
Callback functions are also needed to prevent running into errors because of the non-availability of some data that are needed by some other function.
